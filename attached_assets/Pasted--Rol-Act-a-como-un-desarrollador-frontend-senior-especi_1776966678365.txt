# Rol
Actúa como un desarrollador frontend senior especializado en diseño 
minimalista moderno y experiencia de usuario. Vas a construir un sitio 
web de una cafetería real.

# Objetivo
Crear un sitio web estático, responsive y listo para producción para 
"DailyShot Café", una cafetería de especialidad en Guadalajara, México.

# Stack técnico (obligatorio)
- React 18 + Vite (NO Next.js, NO Create React App)
- Tailwind CSS para estilos
- React Router v6 para navegación entre secciones (SPA con scroll suave)
- lucide-react para iconos
- El proyecto DEBE funcionar con `npm run build` y desplegarse en Vercel 
  como SPA sin configuración adicional
- Todo el código en un solo proyecto, estructura limpia en /src

# Sistema de diseño
Colores (usa estos valores exactos en tailwind.config.js):
- bg-base: #FAFAF7 (blanco cremoso)
- ink: #1A1A1A (negro suave)
- muted: #6B6B6B (gris texto secundario)
- accent: #C97B3C (ámbar cálido, color de marca)
- accent-soft: #F4E4D1 (ámbar muy claro para fondos sutiles)
- line: #E8E5DE (líneas y bordes)

Tipografías (importar de Google Fonts):
- Display/Títulos: "Fraunces", serif (pesos 400, 600)
- Body: "Inter", sans-serif (pesos 400, 500, 600)

Reglas de diseño:
- Espaciado generoso (py-20 md:py-32 entre secciones)
- Max-width del contenido: max-w-6xl mx-auto
- Bordes redondeados sutiles (rounded-lg, nunca rounded-full excepto botones)
- Sin sombras pesadas; usa líneas finas y contraste tipográfico
- Animaciones muy sutiles (hover, fade-in), NO carruseles giratorios
- Mobile-first, perfectamente responsive
